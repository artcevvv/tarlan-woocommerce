
# Tarlan Payments module for WordPress - WooCommerce

## Установка

1. Скопируйте папку `tarlan-payments-gateway` в директорию `wp-content/plugins/` на сервере или установите плагин через раздел плагинов WordPress.

2. Зайдите в "Управление сайтом" -> "Плагины". Активируйте плагин "Tarlan Payments".

3. В админке зайдите в "WooCommerce" -> "Настройки" -> "Платежи" -> "Tarlan Payments". Отметьте галочкой включить данный плагин.

## Настройка

Через "Управление" заполните все поля. Вам нужно будет получить от нас:
1. Идентификатор мерчанта
2. Идентификатор проекта
3. Секретный ключ проекта
4. Тестовый идентификатор мерчанта
5. Тестовый идентификатор проекта
6. Тестовый секретный ключ проекта

Поле "Тестовый режим" помогает протестировать на нашем тестовом API и провести платеж.

Наш официальный сайт: [tarlanpayments.kz](https://tarlanpayments.kz/)
